# __PHASE 1__: Build A Real, Usable Mechanic APP

## __Step 1__ -- Phase 1 feature set
### __Phase 1 includes:__
* One User (no login)
* Manual VIN entry
* Vehicle list
* Vehicle detail page
* Job creation
* Start/stop job timer
* Local storage only

### __Phase 1__ does __NOT__ include:
* VIN decoding
* Cloud sync
* Parts inventory
* Photos
* Multiple Users
